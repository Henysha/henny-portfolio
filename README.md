# Henny Yared Portfolio

![CI](https://github.com/Henysha/henny-portfolio/actions/workflows/ci.yml/badge.svg)

Personal software engineering portfolio for Henok "Henny" Yared.

**Live site:** https://henny-portfolio.vercel.app

This site highlights projects focused on fintech infrastructure, backend APIs, ledger systems, risk engines, funding workflows, and full-stack systems.

## Featured Projects

- **NebirroAPI** — Programmable financial infrastructure API for trusted capital networks.
- **TrustGraph Engine** — Explainable fintech risk, fraud, and reputation engine.
- **CapitalCircle** — Community capital and pooled finance platform.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- lucide-react
- Vercel

## Site Features

- Responsive one-page portfolio layout
- Featured project cards with GitHub links
- Technical skills grouped by engineering area
- Engineering focus, education, and contact sections
- Resume link support
- Custom SVG favicon
- SEO, Open Graph, and Twitter metadata
- GitHub Actions CI for linting and production builds

## Local Development
